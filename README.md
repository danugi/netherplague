# Nether Plague (Fabric)

Мод для Fabric **1.21.11**: незер-заражение распространяется из блоков портала и превращает блоки мира в незеррак.

## Что делает мод

- Каждые 20 тиков мод ищет блоки `nether_portal` рядом с игроками.
- От каждого найденного портала выполняется несколько лучей заражения.
- Первый подходящий блок на луче заменяется на `netherrack`, а огонь сверху появляется с шансом 5%.
- Если луч упирается в `crying_obsidian`, распространение по этому направлению полностью останавливается (барьер действительно режет луч).
- Первый подходящий блок на луче заменяется на `netherrack`, а сверху может появиться огонь.
- Если луч упирается в `crying_obsidian`, распространение по этому направлению полностью останавливается.
- Заражённые блоки (появившиеся из распространения) можно добыть только `netherite_pickaxe`.

Итог: барьер из плачущего обсидиана вокруг портала блокирует заражение.

## Как скачать проект

```bash
git clone <URL_ЭТОГО_РЕПО>
cd netherplague
```

## Как собрать `.jar`

> Требуется Java 21+.

### Linux / macOS

```bash
./gradlew build
```

### Windows (PowerShell)

```powershell
.\gradlew.bat build
```

После сборки готовый файл будет в папке:

- `build/libs/`
- обычно файл вида `netherplague-1.0.0.jar`

## Как установить мод в Minecraft

1. Установи **Fabric Loader** для Minecraft **1.21.11**.
2. Запусти игру один раз, чтобы создалась папка `mods`.
3. Положи собранный `jar` в папку `mods`:
   - Windows: `%appdata%/.minecraft/mods`
   - Linux: `~/.minecraft/mods`
   - macOS: `~/Library/Application Support/minecraft/mods`
4. Запускай профиль Fabric 1.21.11.

## Как протестировать в игре

1. Создай тестовый мир (лучше `Creative`).
2. Построй и активируй адский портал.
3. Рядом поставь обычные блоки (камень/земля/дерево) — они должны постепенно заражаться.
4. Сделай стену/кольцо из `crying_obsidian` между порталом и этими блоками.
5. Проверь, что за барьером заражение не проходит.

Для ускоренной проверки можно выполнить в чате (с включёнными читами):

```mcfunction
/give @p crying_obsidian 256
/give @p obsidian 256
/give @p flint_and_steel 1
```
